# Материалы по изучению китайского языка

## Иероглиф: слог, или слово?

Похоже на то, что у каждого отдельного иероглифа есть собственное значение, а объединение нескольких иероглифов создаёт новое значение, либо уточнение. Возьмём отдельные распространённые иероглифы:

- 你 = Nǐ = You, Ты, тебе
- 不 = Bù = Not, отрицание
- 好 = Hǎo = Good, хорошо
- 错 = Cuò = wrong, mistake, bad
- 再 = Zài = Again, снова
- 见 = jiàn = See, видеть
- 坐 = zuò = Sit, сидеть

И вот примеры комбинаций этих иероглифов:

- 你好 -> Тебе хорошо -> Добра тебе! -> Привет!
- 再见 -> Видеть снова -> Увидимся снова! -> Пока!
- 不错 -> Не плохо -> Нормально!

Таким образом, зная значения отдельных иероглифов можно дешифровать смысл нескольких идущих подряд иероглифов, но лучше запомнить наиболее распространённые комбинации.

## Числа

Числа от нуля до десяти:

- 0 = líng = 零
- 1	= yī = 一
- 2	= èr = 二
- 3	= sān = 三
- 4	= sì = 四
- 5	= wǔ = 五
- 6	= liù = 六
- 7	= qī = 七
- 8	= bā = 八
- 9	= jiǔ = 九
- 10 = shí = 十

Для каждого десятичного разряда есть свой собственный иероглиф (счётное слово):

- 百 = yībǎi = сто
- 千 = qiān = тысяча
- 万 = wàn = десять тысяч

Дальше идут комбинации десятичных разрядов:

 - 十万 = сто тысяч
 - 百万 = миллион
 - 千万 = десять миллионов
 - 亿 = yì = сотни миллионов (!?)

Для обозначения денег используются слова:

- 民币 = rénmínbì = народные деньги
- 币 = bì = деньги
- 钱 = qián = деньги, монеты

Для обозначения мажорных и минорных частей суммы (условно, рублей и копеек), используются счётные слова. При этом есть официальные и разговорные счётные слова.

- 元 = yuán = юань (официально); 块 = kuài = юань (разговорное обозначение)
- 角 = jiăo = цзяо;	毛 = máo = 1/10 yuán = 10 fēn
- 分 = fēn = фэнь; 分 = fēn	= 1/100 yuán = 1/10 jiăo (máo)

Если мы говорим о крупных суммах, то используются правила написания чисел. Например: `一万块` означает `десять тысяч юань`.

Валюты других стран:

- 欧元 = евро
- 美元 = доллары
- 卢布 = lúbù = рубли
- 戈比 = gēbĭ = копейки

Часто используемые выражения:

- 多少钱？= Сколько стоит?
- 一共 = всего
- 不行 = Bù Xíng = Not OK

## Счётные слова

В китайском языке очень широко используются иероглифы, называемые **счётными словами**. Такое слово указывает на тип объекта, используемого в перечислении: человек, животное, бутылка, и т.д.

Наиболее распространённые счётные слова:

- 一个人 = Yī **Gè** Rén. Счётное слово 个 - универсальное, 人 - человек
- 一只猫 = Yī **Zhǐ** Māo. Счётное слово 只 - птица, маленькое животное (кроме собак)
- 九块钱 = Jiǔ Kuài Qián = 块 - деньги, кусок
- 两盘 = Liǎng **Pán** = Две тарелки
- 三杯水 = Sān **Bēi** Shuǐ. Счётное слово 杯 - стакан (с какой-то жидкостью), затем обычно следует иероглиф обозначающий конкретный тип жидкости
- 四瓶可乐 = Sì **Píng** Kělè = 瓶 - бутылка/бутылок
- 这件外套 = Zhè **Jiàn** Wài Tào = 件 - вещь, часть одежды, (кроме юбок и брюк)
- 五条狗 = Wǔ Tiáo Gǒu = 条 - Рыбы, собаки, реки, улицы, юбки и брюки
- 四口人 = Sì Kǒu Rén = 口 - члены семьи
- 一位老师 = Yī Wèi LǎoShī = 位 - уважаемый
- 三顶帽子 = Sān Dǐng Màozi = 顶 - головной убор
- 十本书 = Shí Běn Shū = 本 - Предметы с переплётом (книги, тетради)
- 五台电视 = Wǔ Tái DiànShìTái = 台 - Электрические приборы
- 七把椅子 = Qī Bǎ YǐZi = 把 - Предметы с ручками, стулья
- 五张桌子 = Wǔ Zhāng ZhuōZi = 张 - Предметы с плоскими поверхностями (кровати, столы)
- 三支笔 = Sān Zhī Bǐ = 支 - Длинные вытянутые тонкие предметы (палки, ручки)
- 两幅画 = Liǎng Fú Huà = 幅 - Картины, полотна
- 四辆汽车 = Sì Liàng QìChē = 辆 - Машины
- 九家商店 = Jiǔ Jiā ShāngDiàn = 家 - Компании, отели, рестораны, магазины
- 一双鞋 = Yī Shuāng Xié = Парные предметы

Счётные слова необходимо обязательно заучить.

## Слова, связанные со школой (学 - study, learn)

- 学生 = Xué shēng = student, pupil
- 中学生 = Zhōng Xué shēng = middle school student
- 学校 = Xué xiào = school
- 学习 = Xué xí = studying
- 学问 = Xué wèn = learning, knowledge
- 同学 = Tóng xué = classmate
- 大学 = Dà xué = college
- 放学 = Fàng xué = After school

- 大学 的 时候 = in college

Ключевой иероглиф - 学, т.е. указывает на отношение к школе.

大学校 = Dà Xué xiào = college

Слова `大学校` наглядный пример того, как в китайском языке происходит формирование новых понятий - за счёт комбинирования и добавления уточняющих иероглифов. 学校 = школа + 大 = большой.

## Часть дня (午 - часть дня)

- 上午 = Shàng wǔ = morning
- 中午 = Zhōng wǔ = noon
- 下午 = Xià wǔ = afternoon

Но:

- 早上 = Zǎo shang = morning
- 晚上 = Wǎn shàng = evening, night; 晚 = Late

Применения:

- 上班 = Shàng bān = идти на работу
- 下班回家 = Xià bān huí jiā = идти с работы домой

Иероглифы 上 (Shàng) и 下 (Xià) очень часто используются для обозначения чего-то до и после. Например:

- 上午 = Shàng wǔ = можно перевести как "до дня", т.е. утро
- 下午 = Xià wǔ = можно перевести как "после дня", т.к. вечер
- 上班 = Shàng bān = можно перевести как "идти на работу"
- 下班 = Xià bān = можно перевести как "возвращаться с работы"

## Глаголы

- 回 = Huí = идти отратно, возвращаться
- 给 = Gěi = give, supply, давать
- 买 = Mǎi = покупать
- 去 = Qù = go, идти
- 游 = Yóu = go, walk, travel
- 说 = Shuō = говорить
- 问 = Wèn = спрашивать
- 看 = Kàn = видеть, читать, смотреть
- 用 = Yòng = использовать. Часто переводится по схеме: "мы используем что-то для чего-то"
- 找 = Zhǎo = try to find, looking for. Обычно "я ищу кого-то, или что-то"
- 可以 = Kě Yǐ = can, обычно "ты можешь прийти/сделать что-то"
- 来 = Lái = come, приходи
- 进来 = Jìn lái = come in, входите

## Частота (часто = 常)

- 常常 - Cháng cháng = often, frequently, usually
- 非常 - Fēi cháng = very frequently
- 不常 - Bù cháng = infrequently

## Прочее

- 帅 = shuài = предводитель, видный, солидный. Handsome
- 大 = Dà = large, big, great, major
- 中 = Zhōng = middle
- 小 = Xiǎo = small
- 班 = Bān = class/урок, duty/служба
- 男生 = Nán shēng = boy
- 男朋友 = Nán péng yǒu = boyfriend
- 朋友 = Péng yǒu = friend
- 女朋友 = Nǚ péng yǒu = girlfriend
- 女儿 = Nǚ'ér = daugther
- 儿子 = Ér zi = son

Иероглиф 男 указывает на мужской пол, а 女 - на женский. 朋友 означает "друг". 儿 - суффикс для существительных, в сочетании с иероглифами обозначающими пол, может означать _youngster_ (молодой). Иероглиф 子 означает son, child, seed.

- 师 = Shī = division, разделение, master, expert
- 老 = Lǎo = old, aged
- 老师 = Lǎo shī = teacher

Иероглиф 妈 (Mā) является составной частью слова "мама" (妈妈) и маркером, указывающим на то, что речь идёт о женщине, или даже скорее о замужней, или старшей женщине:

姑妈 = Gū mā = aunt
