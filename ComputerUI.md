# Термины, используемые в пользовательском интерфейсе компьютера

Термины из приложения OpenixSuit:

- 设置 = Shè zhì = Settings, выбор настроек
- 语言 = Yǔ yán = Language, выбор языка
- 中文 = Zhōng wén = Китайский язык
- 简体中文 = Jiǎntǐ zhōngwén = Simplified Chinese language
- 回 = Huí = Return, Go back

Термины из утилиты RKDevTool

- 开 = Kāi = Open
- 驱动 程序 安装 = Driver Installation = Qūdòng chéngxù ānzhuāng
- 肯定 = Sure = Kěn dìng
- 下载 = Download
- 固件 = Firmware
- 下载 镜像 = Download Image = Xiàzài jìngxiàng
- 升级 固件 = Upgrade Firmware = Shēngjí gùjiàn
- 高级 功能 = Advanced Functions = Gāojí gōngnéng
- 存储 = Storage
- 切换 存储 = Switch Storage = Qiēhuàn cúnchú
- 获取 当前 存储 = Get Current Storage = Huòqǔ dāngqián cúnchú
- 切换存储成功 = The storage switch was successful = Qiēhuàn cúnchú chénggōng
- 添加 = Add ... = Tiān jiā
- 删除 = Delete ... = Shānchú
- 清除所有项 = Clear all items = Qīngchú suǒyǒu qǐng
- 上移 = Move up = Shàng yí ... На предыдущий элемент
- 下移 = Move down = Xià yí ... На следующий элемент
- 导入 配置 = Import Configuration = Dǎorù pèizhì
- 导出配置 = Export Configuration = Dǎochū pèizhì
- 强制按地址写入 = Force writing "by address"
- 执行 = Implement / Execute = Zhíxíng
- 开始 切换 存储 = Start Switching Storage
- 开始下载 = Start Downloading
- 下载完成 = Download Completed
